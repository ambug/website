Հայաստանի BSD օգտագործողների խումբ
( AMBUG )

Armenia BSD User Group֊ը նախատեսուած է BSD հաքերների, Unix ադմինների եւ Unix֊անման համակարգեր օգտատէրերի համար։
Մենք (փորձելու ենք) հաւաքուել ամէն ամիս` միասին նորութիւններով ու գիտելիքներով կիսուելու նպատակով։
Միացի՛ր, եթէ լսելու եւ ասելու բան֊ման ունես։
Նախկին եւ ապագայ հանդիպումների մանրամասները ներքեւում։
յ.գ. իհարկէ բոլոր հանդիպումների ժամանակ լինելու են պիցցա, կպչուն պիտակներ եւ հաւէս մարդիկ :-)
